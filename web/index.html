Features

Have you ever read or written a text in a foreign language? Have you ever seen a video in a non-native language to you? Do you know the moment when you have to stop working, pause the video or switch to a translation page because of this one word you don't know how to translate? That's when you fall in love with NinjaTranslate. Invisible when you're good - fast when you need it. Just like a ninja :)

  • Minimalistic design, nothing what could distract you in any way
  • Translations no matter what program you are currently using
  • Easy to extend, quick change key to switch to a different language as fast as possible

Install

All you need to avoid breaking your workflow is NinjaTranslate and a fitting dictionary file. A fitting dictionary file is simply a file containing one word per line, followed by a tab and the fitting translation. You can write it on your own, but you can also download a pre built one.

  • If necessary: Install .Net Framework 4.5 or above
  • Get yourself a fitting dictionary, for example here
  • Start NinjaTranslate, show it the dictionary and specify a name. Don't worry if it takes some time to start, next time it will be a lot faster.
  • NinjaTranslate is now ready to be used

Note: For now NinjaTranslate is limited to Windows only!

Usage

Once it has a dictionary, you can use NinjaTranslate. Don't worry if you can't see it - thats why it is so great. You will find a small icon in the right bottom of your screen. Click on it, and you will be able to set some preferences. You will also be able to add more dictionaries. You can specify which of it should be loaded when NinjaTranslate starts, as well as mark another dictionary as "quick change key". Using this, you can switch between both dictionaries very fast using the quick change key. Once you have finished your preferences, just close the window and do the following:

  • Press Ctrl + Alt + B: A small input window will appear. Enter the word you want to translate and press enter.
  • Press Ctrl + Alt + V: Quick change key, NinjaTranslate will load the dictionary you have marked as quick change (or back to the one you had before). This is useful to translate words in both directions (e.g. ger - en and en - ger)
  • Beta: Mark the word in your text which you want to translate. Press Ctrl + Alt + V: NinjaTranslate will try to detect the text and present the translation.

Once you have entered a word to translate, a small window will show you the translations. You can specify the size of this window. Use any button to hide the window again. With a little training, you will be able to translate words without interrupting your work.
